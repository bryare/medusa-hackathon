![Medusa Hackathon 2022](/medusa.jpg)

1. Medusa Server on localhost:9000
   <br/>
   npm install @medusajs/medusa-cli -g
   <br/>
   medusa new medusa-server --seed
   <br/>
   cd medusa-server
   <br/>
   medusa develop

2. Medusa Store on localhost:8000
   <br/>
   cd medusa-store
   <br/>
   yarn install
   <br/>
   npm run dev
